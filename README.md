# StayMemo

  <img src="icon.png" width="128" alt="StayMemo icon">
</p>

macOSのメニューバーに常駐するシンプルなメモアプリです。

## 特徴

- メニューバーのアイコンをクリックで即座に開閉
- 3ページのメモを切り替えて使用
- テキストは即時保存、再起動後も維持
- ピン留め機能で常に最前面に固定
- プレーンテキスト専用（書式付きペーストも自動でプレーン化）
- フォント設定可能（右クリックメニュー → 設定）
- ウィンドウの位置・サイズを記憶
- ローカル完結、ネットワーク不要
- **非表示文字の可視化**（Invisible Prompt Injection 対策）

## 非表示文字の可視化

メモ画面のツールバーにある `¶` ボタンで、テキスト中の非表示文字をバッジ表示に切り替えられます。LLMへのコピー&ペースト前に Invisible Prompt Injection が仕込まれていないか確認するのに役立ちます。

| バッジ | 意味 |
|--------|------|
| `·` | スペース（通常・ノーブレーク・各種幅違いスペース等） |
| `→` | タブ |
| `¶` | 改行（LF） |
| `↵` | 復帰（CR） |
| `∅` | ゼロ幅文字・完全不可視文字（U+200B/C/D、U+FEFF 等） |
| `⇄` | 方向制御文字（RTL オーバーライド U+202E 等） |
| `A`〜`z` 等 | Unicode Tags ブロック（U+E0000–U+E007F）の隠しテキストをデコード表示 |

表示色は右クリックメニュー → 設定 から変更できます。

## ショートカットキー

| キー | 操作 |
|------|------|
| `⌘1` / `⌘2` / `⌘3` | ページ切り替え |
| `⌘P` | ピン留めのON/OFF |

## 動作環境

- macOS 14 (Sonoma) 以降

## インストール

[Releases](https://github.com/Shinichi-Ohki/stay_memo/releases) から `StayMemo.app.zip` をダウンロードして展開してください。

署名なしアプリのため、初回実行前に以下のコマンドで隔離属性を解除する必要があります:

```bash
xattr -rc StayMemo.app
```

## ビルド方法

Xcode GUIは不要です。コマンドラインでビルドできます。

```bash
# デバッグビルド
swift build

# リリースビルド + .appバンドル作成
./build.sh

# 実行
open StayMemo.app
```
